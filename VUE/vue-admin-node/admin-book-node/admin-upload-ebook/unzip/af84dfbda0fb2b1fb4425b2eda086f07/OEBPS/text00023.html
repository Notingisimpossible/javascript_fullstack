后记
正在发生的未来

2012年9月，我以前的同事杨锐来上海看望我。那天，我们从上午聊到下午，整整聊了6个小时，话题因我案头摆放的几本关于云计算、大数据、社交网络的书而起。当然，我们说到最多的，是移动互联时代，投资管理乃至资产管理行业将会受到的冲击，并随之会产生的变化。窗外黄浦江水奔流向海，浩浩荡荡，滔滔不绝且不可阻挡。我们6个小时的讨论对这些话题而言，只是匆忙地开了个头。

转眼到了2013年5月，互联网金融在神州大地已如初夏的热潮。我给杨锐打电话：“我们应该就互联网金融写点什么了，一则算是纪念我们曾经的资产管理从业生涯，二则算是系统梳理我们近一年来的学习与实践心得。”

写这样一本关于未来的书是十分吃力，也许还并不讨好的事。我的心路历程确实也是一波三折：从一开始因体会与触动而起缘写书；进而因梳理其中的层次、结构和逻辑而反复折腾；到最后因书成而坚信这一切就是正在发生的未来。我不能确定，是我和我的小伙伴们成就了这本书，还是这本书成就了我们。

本书想要表述的第一个主题是：移动互联网、云计算、大数据将颠覆资产管理行业。任何关于移动互联网、云计算、大数据将颠覆资产管理行业的论述和预测，在余额宝的成功实践面前，都显得多余了。德国伟大诗人歌德有一句名言：理论是灰色的，而生命之树常青。此情此景，岂不令人唏嘘。

本书想要表述的第二个主题是：在知识经济时代，基金经理们的投资方法也必须与时俱进；一是比特经济的估值方法与原子经济的估值方法必然不同；二是社交网络和大数据将成为观察市场的新利器。如果说少数基金经理的“老鼠仓”行为是腐败，那么不能在投资方法上与时俱进的基金经理就是腐朽。腐朽与腐败一样有害。

本书想要表述的第三个主题是：互联网乃至物联网使得客观世界与主观世界融通，现实世界与虚拟世界融合，线上世界与线下世界融汇，原子世界与比特世界融解。工业革命使人类极大地开垦了物质世界财富，而信息革命将使人类极大地开发数字世界的财富。据史学家麦迪森估算，公元1年世界人均GDP按1990年不变价计算为445美元，到1820年英国工业革命时为667美元，1 800多年只增长了50%。而在1820~2001年的180年间，世界人均GDP从667美元增加到6 049美元，增长了8倍多。

加利福尼亚大学格里高利·克拉克教授进一步指出：世界人均GDP在1800年前的几千年里没有什么变化，年均增速几乎可以忽略不计，只有1800年英国工业革命之后，随着世界人均GDP的显著变化，人类的生活方式、社会结构、政治形态、文化内涵才发生了本质性的变化。

我理解，在工业革命之前，人类一直使用畜力和铁器协助生产，而牛马等畜力和镰刀、锄头等冷工具本身进化有限，所以，人类的劳动生产率在1 800年间提升不多。而工业革命极大地解放了人类，蒸汽机、铁路、无线电、计算机、互联网对人类劳动生产率的提升，起到巨大的推进作用。我们正在跨入的信息革命时代，有机会使人的大脑在移动互联网、物联网、云计算、大数据、社交网络等新技术、新工具、新模式的帮助下，得到极大释放，从而提供一个开发人的主观世界的新的历史性机会。

这是自工业革命之后，人类历史上第二次波澜壮阔的信息革命。人类财富的增加值将来自数字世界。在过去10年，无论中国、美国的所谓蓝筹股，回报都几乎可以忽略不计，而代表数字经济的新型公司却贡献了惊人的回报。2014年1月31日，谷歌市值超过埃克森美孚，成为美国市值第二大公司，而苹果市值排名第一。一位分析师评论道：这是时代的信号。传统公司正在退出，新一代公司正在进入。

本书的诞生并不是我一个人的功劳。作者还包括杨锐、王政、魏凤春、赵海健、刘思甸、余军、查晓磊，他们都是我在博时基金工作时的同事，其中几位目前还服务于博时基金。说感谢有点见外，我们是携手共同进入新时代的小伙伴。

我还要感谢中信出版社的蒋永军、王律等编辑，谢谢你们的意见和建议，也谢谢你们团队的辛苦工作。

我正在筹建一家侧重互联网与资产管理方向的互联网金融研究与实验机构，以延续对移动互联时代金融行业发展变化趋势的研究与实验。也许，谦卑的态度、持续的学习、不断的实验，才是我们面对新时代的生存之道。

美国科幻小说作家威廉·吉布森说过：“未来早已到来，只是还未普及。”
